Slack - チームコミュニケーションプラットフォーム

■ 概要

Slackは、チーム向けのリアルタイムメッセージングプラットフォームである。
2013年8月にプレビュー公開、2014年2月に正式リリース。
Stewart Butterfield（Flickr共同創業者）が率いるTiny Speck社が開発。
元々はオンラインゲーム「Glitch」の社内コミュニケーションツールとして開発されたものを
製品化した経緯を持つ。

■ 解決する課題

1. メールの非効率性: CCの乱用、スレッドの追跡困難、リアルタイム性の欠如
2. 社内情報のサイロ化: 部門間の情報共有が困難、知識の属人化
3. 既存チャットツールの使いにくさ: HipChat、IRC、Campfireは開発者向けでUI/UXが貧弱
4. ツール間の分断: メール、チャット、ファイル共有、タスク管理が別々のツール
5. リモートワーク対応: 分散チームのコミュニケーション基盤が未整備

■ 主な機能

基本機能:
- チャンネルベースのメッセージング（パブリック・プライベート）
- ダイレクトメッセージ（1対1、グループ）
- ファイル共有（Dropbox、Google Drive連携）
- 全文検索（メッセージ、ファイル横断）
- スニペット（コード共有）
- スター付きメッセージ
- カスタム絵文字
- 通知のカスタマイズ

差別化機能（2014年時点）:
- 豊富なサードパーティ連携（ローンチ時80+、2014年末に200+）
- Slackbot（カスタマイズ可能なボット）
- 高品質な検索（メッセージ内容、ファイル名、送信者でフィルタ）
- マルチプラットフォーム（Web、Mac、Windows、iOS、Android）
- Incoming/Outgoing Webhooks、Slash Commands

■ 技術スタック（2014年時点）

- フロントエンド: JavaScript（jQuery）、後にReactへ移行
- バックエンド: PHP（一部Java、後にNode.jsも）
- リアルタイム通信: WebSocket
- データベース: MySQL + Memcached
- 検索エンジン: Apache Solr
- インフラ: AWS（us-east-1）
- メッセージング基盤: 独自開発のメッセージバス

■ 価格プラン（2014年時点）

- Free: メッセージ履歴10,000件、連携5つまで、5GBストレージ
- Standard: $6.67/ユーザー/月（年払い）、無制限履歴、無制限連携、10GBストレージ
- Plus: $12.50/ユーザー/月（年払い）、SAML SSO、コンプライアンスエクスポート、20GBストレージ

■ ターゲット市場

第1ターゲット:
- テクノロジー企業（スタートアップ、テック大手のチーム単位）
- ソフトウェア開発チーム（GitHub連携がキラー機能）
- 50〜500人規模の成長企業

第2ターゲット:
- メディア・クリエイティブ企業
- リモートワーク中心の企業
- 大企業内の先進的な部門

■ チーム・資金調達

- 創業者: Stewart Butterfield（CEO）、Cal Henderson（CTO）、Eric Costello、Serguei Mourachov
- 設立: 2009年（Tiny Speck社として、ゲーム開発）
- Slack開発開始: 2012年（ゲーム事業のピボット）
- 資金調達:
  - 2014年4月: シリーズC $4,270万（Accel Partners等）
  - 2014年10月: シリーズD $1.2億（Kleiner Perkins、Google Ventures等）
  - 評価額: $11.2億（2014年10月時点、ローンチ8ヶ月でユニコーン化）

■ 当時の状況と実績（2014年）

- 2014年2月正式ローンチ
- ローンチ24時間で8,000社がサインアップ
- 2014年8月: DAU 17.1万、有料チーム数1.5万
- 2014年11月: DAU 28.5万、有料チーム数3万
- 日次メッセージ数: 2014年末で1日あたり数百万件
- メッセージの送受信量で「世界最速成長のビジネスアプリ」と呼ばれる
- SaaS史上最速で$1M ARRから$12M ARRに到達
- 口コミ主導の成長: 広告費をほぼ使わず、プロダクト品質と口コミで拡大

■ ロードマップ（2014年時点）

- 2014年後半: Windows版デスクトップアプリ正式リリース
- 2014年末: Enterprise向け機能の拡充
- 2015年: APIプラットフォームの強化、サードパーティ連携の大幅拡充
- Slack Fundの設立（サードパーティ開発者支援ファンド、$8,000万規模）
