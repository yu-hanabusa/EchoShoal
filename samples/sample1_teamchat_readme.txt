TeamChat - リアルタイムチームコミュニケーションプラットフォーム

## 概要

TeamChatは、日本の中小企業向けに設計されたチームコミュニケーションツールです。
Slackライクな機能を日本語UIと国内データセンターで提供し、
セキュリティとコンプライアンスを重視する企業ニーズに応えます。

## 主な機能

- チャンネルベースのメッセージング（テキスト、画像、ファイル共有）
- スレッド機能によるトピック別議論
- ビデオ通話・画面共有（最大50人）
- 外部サービス連携（Google Workspace、Microsoft 365、kintone、Backlog）
- AIアシスタント（議事録自動生成、タスク抽出）
- 監査ログ・情報漏洩防止（DLP）機能
- オンプレミス版の提供

## 技術スタック

- フロントエンド: React + TypeScript
- バックエンド: Go + gRPC
- リアルタイム通信: WebSocket
- データベース: PostgreSQL + Redis
- 検索: Elasticsearch
- インフラ: AWS東京リージョン（オンプレミス版はKubernetes対応）

## 価格プラン

- フリープラン: 10人まで無料、メッセージ履歴1万件
- スタンダード: 月額500円/ユーザー、無制限履歴、外部連携
- ビジネス: 月額1,200円/ユーザー、SSO、監査ログ、DLP
- エンタープライズ: 要見積、オンプレミス、専用サポート

## ターゲット市場

- 日本国内の従業員50〜500人規模の企業
- 特に製造業、金融、公共セクターなどセキュリティ要件の厳しい業種
- Slackの英語UIや海外データセンターに不安を感じている企業

## チーム

- 創業者: 元Slack日本法人のエンジニアリングマネージャー
- メンバー: 12名（エンジニア8名、セールス2名、CS2名）
- 設立: 2025年6月
- 資金調達: シード1.5億円（国内VCより）
