# サンプルシナリオ

## Sample 1: TeamChat（ビジネスチャットツール）

日本の中小企業向けチームコミュニケーションツール。Slack対抗で国内データセンター・日本語UIを強みに参入。

### 入力方法

**Service Name:**
```
TeamChat
```

**Scenario:**
```
日本の中小企業向けビジネスチャットツール「TeamChat」が2026年4月に正式リリースされる。Slackライクな機能を日本語UIと国内データセンター（AWS東京リージョン）で提供し、セキュリティ重視の製造業・金融・公共セクターをターゲットに月額500円〜のフリーミアムモデルで参入する。元Slack日本法人のエンジニアが創業し、シード1.5億円を調達済み。ベータ版30社参加でNPSは+45と高評価だが、市場はMicrosoft Teams（シェア17.25%、導入企業内利用率67.8%）が圧倒的に強く、Slack（3万社以上）、LINE WORKS（46万社）、Chatwork（中小企業で45%のレビューシェア）という強力な競合が存在する。ビジネスチャット市場は2025年度446億円規模で成長率は鈍化中。AI議事録機能とISMAP対応のオンプレミス版が差別化要因。
```
